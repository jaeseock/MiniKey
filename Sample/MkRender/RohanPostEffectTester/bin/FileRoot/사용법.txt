0. 툴의 목적
- 로한의 이미지 업그레이드를 위한 설정샷을 만들어 공유합니다.
- 시안들에 따라 해당 규칙과 세팅, 적용 방법등도 논의합니다.
- 모든 사안이 완료되면 이펙트를 최적화 해 게임에 적용합니다.

1. 최초 설치
- 적당한데 새 폴더를 만들고,
- "Updater.exe" 파일을 복사 한 후 실행하면,
- 해당 폴더에 툴이 설치됩니다.
- 바로 실행하시려면 [게임 실행] 버튼을 누르세요.

2. 업데이트
- 최신화된 툴을 받고 싶으시다면 "Updater.exe"를 실행해주세요.

3. 툴 실행
- "RohanPostEffectTester.exe"를 실행합니다.
- 최초 실행시에 장면 이미지들이 저장된 디렉토리 경로를 지정합니다. 나중에 다시 설정 할 수 있으니 취소하셔도 됩니다.
- 장면 선택 창에서 따블클릭으로 원하는 장면을 선택합니다.
- 장면을 고르면 기본적으로 메인 설정 창이 뜹니다. 설정 로딩과 이펙트 개별 적용여부, 설정 창을 열 수 있습니다.
- 설정 샘플은 "Sample" 폴더 안에 있습니다.
- 단축키는 우편 모니터링 윈도우에 나와 있습니다.
	* F2 : 오버레이(원본 -> 깊이버퍼 -> 끔). 세팅 중 원본과 비교해보고 싶을 때 사용합니다.
	* F3 : 모든 창 닫음.
	* F5 : 장면 선택 창 토글.
	* F6 : 메인 설정 창 토글.
- 가지고 노시다 마음에 드는 화면이 나오면 메인 설정 창에서 저장합니다. 설정 파일과 스크린샷이 저장됩니다.
- 툴을 종료하면 직전 설정이 자동으로 저장되고 재실행할때 복구됩니다.
- (주의!!!) 깊이 안개 설정용 1D 이미지는 반드시 "Gradation" 폴더 안의 "Add", "Mult" 폴더 안에 위치해야 합니다.
- 해상도를 바꾸고 싶으면 "AppSetting.txt" 파일을 열어 "Size" 항목의 값을 수정 후 툴을 재실행합니다.

4. 게임 스크린샷 찍기
- DEV 버전 클라이언트가 필요합니다. 없으시면 클라팀에게 강탈합시다.
- 설치 후 패치를 받고 실행합니다.
- 화면 해상도를 (1024, 768) 윈도우 모드로 변경합니다.
- "data" 폴더 안의 "effectrender.mmd" 파일을 텍스트에디터(ex> 메모장)로 열어둡니다.
- 월드를 돌아다니다가 필링이 느껴지는 장소를 발견했다면,
	* 열어 둔 "effectrender.mmd" 파일의 "str Save = "filename";" 항목의 filename 부분을 원하는 이름으로 바꿉니다.
	* 텍스트 파일을 저장합니다.
	* 곧 클라이언트에 "_ess" 폴더가 생성되고 안에 스크린샷이 저장됩니다.
- 돌아다니다 filename을 바꾸고 저장해 마구마구 감성이 충만한 사진을 찍어줍시다.

5. 게임 스크린샷 툴 연동
- 게임 클라이언트 경로가 변경되지 않는다면 한 번만 하면 됩니다.
- 툴을 실행하기 전 "AppSetting.txt" 파일을 삭제합니다.
- 툴을 실행합니다. 장면 이미지들이 저장된 디렉토리 경로를 다시 묻습니다.
- 스크린샷이 저장 된 클라이언트의 "_ess" 폴더를 지정해줍니다. 자동으로 장면이 등록됩니다.
- 한 번 경로가 지정되었으면 이후에는 툴이 시작될때 자동으로 클라이언트의 스크린샷 폴더를 분석해 등록해줍니다.

6. 활용
- 영혼을 뒤흔드는 설정이 나왔으면 세팅파일, 장면 파일(일반, 깊이 이미지 두 장), 안개용 이미지 파일 두 개를 잘 모아놓습니다. 장면이나 안개용 이미지가 기존에 이미 있던 파일이면 따로 모아놓을 필요는 없습니다.
- 해당 파일들을 클라팀에게 전달합니다.
- 곧 해당 설정은 패치되어 툴 사용자 모두에게 공개됩니다.
